# CyberCafeBooking
## Бронирование интернет-кафе

### Структура проекта
```
/root-проекта
|-- backend/                  # Код бэкенда
|   |-- main.go               # Главный файл запуска
|   |-- routes/               # Обработка маршрутов
|   |   |-- booking.go        # Маршруты для работы с бронированиями
|   |-- models/               # Модели данных
|   |   |-- booking.go        # Модель бронирования
|   |-- database/             # Подключение к базе данных
|   |   |-- db.go             # Инициализация MySQL
|   |-- go.mod                # Go зависимости
|-- frontend/                 # Фронтенд
|   |-- index.html            # HTML файл
|   |-- assets/
|       |-- css/
|       |   |-- styles.css    # Стили
|       |-- js/
|           |-- script.js     # Логика взаимодействия
|-- config/                   # Настройки проекта
|   |-- config.json           # Параметры базы данных
|-- README.md                 # Документация

```
